CS Revit Labs Autodesk Revit 2024

An enterprise-grade automation add-in suite developed to streamline model coordination, accelerate construction documentation, enforce graphical quality control, and integrate AI reasoning into BIM environments.

Language & Framework
C# (7.3) .NET Framework 4.8 WPF (XAML)
APIs & Integrations
Revit API 2024 Google Gemini API Newtonsoft.Json PDFsharp
// Architecture
Modeless single-assembly DLL. Employs Costura.Fody to embed all references and dependencies into the C# binary. This simplifies enterprise deployment down to just two files: CSRevitLabs.dll and CSRevitLabs.addin.
